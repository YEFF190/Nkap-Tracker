📱 Nkap Tracker

Nkap — money in Cameroonian slang 💰

A professional mobile money expense tracker built with Flutter, designed specifically for Cameroonians using MTN MoMo and Orange Money. Track your income, expenses, and financial habits — all in one place, completely offline.

✨ Features
💳 Transaction Management

Manually log income and expenses
Categorize transactions (Food, Transport, Bills, School, Business)
Support for both MTN MoMo and Orange Money

📲 SMS Auto-Detection

Automatically reads MTN and Orange Money SMS confirmations
Instantly logs transactions without manual input
No duplicates — each SMS is imported only once

📊 Statistics & Analytics

Visual pie chart of expenses by category
Income vs Expenses bar chart
Savings rate calculator
Monthly spending overview

🔐 Security

Optional PIN lock to protect your data
All data stored locally on your device
No internet connection required

🎨 Professional UI

Clean corporate design
Dark navy blue theme with gold accents
Smooth animations throughout

🚀 Installation
Option 1 — Download APK

Download app-release.apk from the releases section
Enable "Install from unknown sources" on your Android phone
Install and enjoy!

Option 2 — Build from source
bash# Clone the repo
git clone https://github.com/YEFF190/Nkap-Tracker.git
cd Nkap-Tracker

# Install dependencies
flutter pub get

# Run the app
flutter run

📋 Requirements

Android 8.0 (API 26) or higher
SMS permission for auto-detection feature
MTN MoMo or Orange Money account (optional)
